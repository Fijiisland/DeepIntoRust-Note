7.1 简介

简介

模式解构 Pattern Destructure

let tuple = (1_i32, false, 3f32);
let (head, center, tail) = tuple;

这个例子中，将三个元素组合为一个tuple，在第二行代码中，又将tuple解构为三个独立的元素。在let语句中，等号左边即为模式，等号右边即为被解构的对象。在这个例子中，左边的模式引入了三个新变量，分别绑定了tuple的三个成员。

看一个复杂点的例子：

struct T1 (i32, char);

struct T2 {
    item1: T1,
    item2: bool,
}

let x = T2 {
    item1: T1(0, 'A'),
    item2: false,
};

let T2 {
    item1: T1(value1, value2),
    item2: value3,
} = x;

这个例子很好说明“如何组合就如何解构”这个思想。
